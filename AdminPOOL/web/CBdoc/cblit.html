Papers related to ConceptBase

Some of the following papers are available in GNU-gzip-ped Postscript formatted for ISO-A4 paper format. Some WWW browsers will not automatically decompress (un-gzip) the documents. If so then please save the file on disk und decompress it with a proper version of the GNU-gzip program (e.g., WinZIP for Windows PCs, StuffIt w/ Enhancer for Macintosh).

If your printer is accepting only US letter paper format then read the instructions in pub/reports/TROUBLESHOOTING. The ordering of the references is chronological. If you feel that your paper on Telos or ConceptBase is missing then let us know.
