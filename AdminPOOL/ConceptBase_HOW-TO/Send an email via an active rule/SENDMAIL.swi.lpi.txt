/*

* This file is a plug-in for the ConceptBase server
* to allow it to send email to certain users.
* Use of this file requires permission by the author.
* 
* The code below conforms to SWI-Prolog. It uses a some internal
* ConceptBase functions. The code will only run on Linux/Unix
* where the sendmail utility has been installed. You might have to
* adapt the path of sendmail.
*
* You have to copy this file into the database directory where it shall
* be used.
*
* See also README.txt for more details.

/* We assume that mailtext contains no double-quote '"'. Otherwise, the echo command would fail. */ 

sendObject(_object,_email) :-
  cbserver:ask(produceMailText,[_object/whichform,_email/receiver],_mailtext),
  cbserver:concat(['echo "',_mailtext,'" | /usr/sbin/sendmail -t'],_command),
  shell(_command),
  !.

sendObject(_,_) :-
  write('>>> SENDMAIL.swi.lpi: sendObject failed!!!!'),nl,
  !.
