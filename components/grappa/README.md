# AT&T Grappa 1.2

Maven module (`com.conceptbase.legacy:grappa:1.2`) — Java DOT/graph library (`att.grappa`),
used by ConceptBase graph tooling. Parent POM: `../legacy/pom.xml`.

## Build

```bash
mvn install
```

Or build the full Java reactor from `components/java/` (includes this module).

## Source provenance

- Recovered from [phenotips/grappa](https://github.com/phenotips/grappa) (fork of AT&T Grappa).
- Maven coordinates kept as `com.conceptbase.legacy:grappa:1.2` for compatibility with the
  previous `grappa1_2.jar` binary.
